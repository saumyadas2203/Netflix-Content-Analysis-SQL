# Netflix-Content-Analysis-SQL
📊 Netflix Content Analysis Using SQL
📌 Project Overview

This project performs a comprehensive exploratory data analysis (EDA) on the Netflix dataset using advanced SQL techniques. The goal is to uncover actionable insights about content distribution, genre trends, country-wise production, ratings, and yearly growth patterns. This simulates real-world streaming platform analytics and demonstrates the ability to derive business insights from structured data.

🛠 Tools & Technologies

SQL (MySQL / PostgreSQL)

Dataset: Netflix Titles CSV

Querying techniques: JOINs, CTEs, Subqueries, Window Functions, Aggregations

📂 Dataset Description

The dataset includes detailed information on Netflix titles:

Title

Type (Movie / TV Show)

Genre

Release Year

Rating

Country

Duration

Date Added

📈 Key Analyses

Distribution of Movies vs TV Shows

Popular genres and content trends

Country-wise production analysis

Year-wise content growth patterns

Rating classification breakdown

Top longest movies and TV shows

Trends in content addition post-2015

🧠 SQL Skills Demonstrated

Advanced SELECT queries with JOINs and CTEs

Aggregations, GROUP BY, and ORDER BY

Subqueries and CASE statements

Window functions for ranking and trend analysis

Data-driven insight generation

🔍 Key Insights

Significant content growth after 2015

Movies dominate overall library; Drama and International content are highly represented

USA, India, and UK lead in content production

Viewer-targeted genre expansion observed over years
